Detailed Description

Constraint is the abstract class giving a generic interface of constraints. This class is generic, it needs to know both the Variable and the SetVariables types it will work with.

abstract Dictionary<int, double> ghost.Constraint< TypeSetVariables, TypeVariable >.SimulateCost ( int  currentVariableIndex,
Dictionary< int, double[] >  variableSimCost 
)
pure virtual

This function is called by the solver when it looks for a new value for a selected variable (a variable the solver heuristics choose to modify in priority). Given this variable, SimulateCost must try all values in the variable domain, computes the new constraint cost and saves the couple (value, cost) into a Dictionary.

Parameters
currentVariableIndexis the index of variable to change the value.
variableSimCostis a Dictionary<int, double[]>. Its purpose is to save the cost of each variable in the set, and this for each value in the domain of the variable at the index currentVariableIndex.
Returns
A Dictionary<int, double> containing the cost for each value in the domain of the variable at the index currentVariableIndex.
