\subsection{Detailed Description}
Null\-Objective is an empty objective function, useful if one is only looking to solve a satisfaction problem (i.\-e., return the first solution the solver can find among all). The name of such a fake objective will be 'null\-Objective'. Cost will always return 0.\-0. \begin{Desc}
